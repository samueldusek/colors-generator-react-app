## To run this app

Clone the project

```bash
  git clone https://github.com/samueldusek/colors-generator-react-app.git
```

Go to the project directory

```bash
  cd colors-generator-react-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
